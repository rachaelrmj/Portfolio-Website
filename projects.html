Trip Snapshot

A responsive travel planning website for organizing trips, exploring destinations, and creating an itinerary tailored to the individual user's needs. This project was created in order to deal with the frequent issue that the majority of tourists face when using different websites to explore their destinations.

Make a responsive travel planning website for organizing trips, exploring destinations, and creating an itinerary tailored to the individual user's needs. It addresses the frequent issue that the majority of tourists face when using different websites to explore their destinations.

The concept of Trip Snapshot has been generated from ideas drawn from modern travel apps like Airbnb, TripAdvisor, and Google Travel. It merges destination exploration, trip planning, and trip organization into one platform. The design focuses on simplicity, easy navigation, and mobile usability.

HTML, CSS, and JavaScript languages were utilized for designing the site following the principle of mobile-first design. The responsive design of the site was realized with the help of Flexbox, CSS Grid, and media queries in order to ensure cross-platform adaptability of the website. Integration with APIs was utilized for obtaining information about destinations and relevant travel-related data. Technologies of storing information about trips in the browsers' memory were applied, including sessionStorage and localStorage.

The project delivers a full-fledged platform for travel planning where the user can search for destinations, arrange information regarding the trip, and plan his/her itineraries accordingly. It showcases my capability of implementing responsive designs, APIs, and accessibility along with front-end development.

8o5 Cinema

A responsive web application that simulates a modern movie theater reservation experience. The project aimed to create an engaging platform where users could browse movies, view trailers, explore showtimes, and interact with theater services while demonstrating advanced front-end development techniques.

Develop a responsive web application that simulates a modern movie theater website experience. The project aimed to create an engaging platform where users could browse movies, view trailers, explore showtimes, and interact with theater services while demonstrating advanced front-end development techniques.

The design idea was borrowed from commercial theater sites like AMC Theaters and Regal Cinemas. The website uses common entertainment industry designs such as showcasing movies and interactive trailers. The aim was to recreate the actual functions of a theater site but in an easy to navigate format.

The website utilizes HTML, CSS, JavaScript, and jQuery for development. There are many interactive features such as an image slider, modal windows, movie listings and trailers, to allow for a higher degree of user engagement. To create an interactive, dynamic application, external APIs were used along with browser storage to store user's preferences and memberships. Usability and interface design were important in the development of the application.

The website successfully simulates a live entertainment setting. The following were among the aspects that the website was able to demonstrate: proficient use of APIs, jQuery programming, responsive design, and creating an interactive user interface.

Sapphire Dress

A static fashion boutique website built with HTML and CSS that demonstrates semantic markup, visual design, navigation, and foundational front-end web development principles.

Design and develop a visually appealing one-page website for a fashion boutique that showcases products and brand identity through a clean and modern web experience.

The website is inspired by current fashion brands and boutique websites that use visual appeal to draw their consumers. A focus has been made on typography, visuals, hierarchy, and branding. This website tries to examine how good design can help convey the essence of a brand without any functional complexity.

The site was created using HTML and CSS, keeping in mind the principles of visual communication. Single page navigation was chosen to provide a clean browsing process while still providing an organized flow of information. CSS techniques have been utilized to make sure there is consistency regarding space, typography, color, and layout.

A successfully developed website showcasing the products and identity of the boutique was achieved with the use of an elegantly-designed one-page layout.
